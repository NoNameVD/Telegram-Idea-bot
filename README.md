# Telegram Idea Bot

Telegram-бот, который генерирует идеи для создания проектов с помощью LLM (OpenRouter).

## Возможности
- Генерация идей проектов по запросу пользователя
- Краткое описание к каждой идее
- Асинхронная работа на aiogram

## Что нужно для запуска
1. API-ключ Telegram-бота
2. API-ключ LLM (OpenRouter)
